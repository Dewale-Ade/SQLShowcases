This project focuses on cleaning, standardizing, and enriching a raw sales dataset provided as part of the MeriSKILL Internship Program.
The objective was to transform unstructured transactional data into a reliable, analysis-ready dataset suitable for business intelligence, reporting, and decision-making.

All transformations were performed directly in PostgreSQL using SQL, following best practices for data quality and reproducibility.

Tools & Technologies

PostgreSQL

SQL (CTEs, window functions, CASE statements)

pgAdmin

Excel (initial data inspection only)
